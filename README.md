# CarbonChain

CarbonChain is a Carbon Credit Exchange Platform connecting NGOs/SHGs, MNCs, and the NCCR (National Carbon Credit Registry) for transparent carbon credit transactions secured by blockchain technology.

**Live Demo:** [Click Here to View App](https://hasan-dhanish.github.io/carboncredit/login.html)

## Overview

The platform facilitates the trading of carbon credits between organizations generating them (like NGOs and SHGs involved in reforestation, renewable energy, etc.) and corporations (MNCs) looking to offset their carbon footprint. The entire process is overseen by the NCCR to ensure compliance and authenticity.

## Features

-   **Role-Based Dashboards:** tailored interfaces for NGOs, MNCs, and NCCR Admins.
-   **Blockchain Security:** Ensures transparent and immutable transaction records.
-   **AI Validation:** Utilizes satellite imagery and Random Forest AI algorithms to validate carbon sequestration reports (simulation).
-   **Marketplace:** A hub for buyers to browse and purchase credits from verified projects.
-   **Portfolio Management:** Tools for users to track their credits, investments, and impact.
-   **Reporting & Analytics:** Comprehensive reporting tools for NGOs and oversight analytics for Admins.

## User Roles

1.  **NGO/SHG (Sellers):**
    -   Submit carbon sequestration reports for verification.
    -   Manage active projects (Reforestation, Solar, etc.).
    -   View generated credits and revenue.
2.  **MNC (Buyers):**
    -   Browse the marketplace for carbon credits.
    -   Purchase credits to meet offset targets.
    -   Track portfolio and expiry of credits.
3.  **NCCR Admin (Regulators):**
    -   Oversee the entire platform.
    -   Monitor transactions and verify reports.
    -   Manage organizations and audit logs.

## Tech Stack

-   **Frontend:** HTML5, CSS3 (Custom properties/Variables), JavaScript (Vanilla)
-   **Icons:** FontAwesome 6.4.0
-   **Design:** Responsive layout with a modern, clean UI utilizing distinct color themes for different roles.

## Getting Started

1.  Clone the repository.
2.  Open `login.html` in your web browser.
3.  Select a user role (NGO, MNC, or NCCR) to login (simulated authentication).

## Project Structure

-   `login.html`: Entry point with role selection.
-   `ngo-dashboard.html`: Dashboard for NGOs/SHGs.
-   `mnc-dashboard.html`: Dashboard for MNCs.
-   `nccr-dashboard.html`: Dashboard for the NCCR Admin.
